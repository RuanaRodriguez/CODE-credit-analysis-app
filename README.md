# Sistema de Análise de Crédito

Sistema modernizado para solicitação, análise e aprovação de crédito.

## 🚀 Tecnologias

- React 18
- Material-UI
- React Router v6
- React Hook Form
- Recharts
- Vite

## 📦 Instalação

```bash
npm install
```

## 🔧 Desenvolvimento

```bash
npm run dev
```

Acesse: http://localhost:3000

## 🏗️ Build

```bash
npm run build
```

## 📁 Estrutura do Projeto

```
src/
├── components/     # Componentes reutilizáveis
├── pages/          # Páginas da aplicação
├── services/       # Mock services
├── types/          # Estruturas de dados
├── data/           # Mock data
├── utils/          # Utilidades
├── hooks/          # Custom hooks
├── contexts/       # React contexts
└── styles/         # Estilos globais
```

## 🎯 Funcionalidades

- ✅ Criação de solicitações de crédito
- ✅ Validação CPF/CNPJ em tempo real
- ✅ Análise automática de score
- ✅ Workflow de aprovação por alçada
- ✅ Dashboard operacional
- ✅ Histórico de análises
- ✅ Relatórios e métricas
